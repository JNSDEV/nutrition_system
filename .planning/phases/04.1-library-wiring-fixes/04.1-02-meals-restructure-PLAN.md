- H2 = archetype category: `## Breakfast`, `## Lunch`, `## Dinner`, `## Snacks`
- H3 = meal name (Title Case, verbatim from the existing bullet text, with minor normalisation for anchor clarity)
- Under each H3, immediately: `**archetype:** breakfast` (or lunch / dinner / snack — matching the H2 it sits under)

- Remove leading "Leftover " prefix — anchor consumers look up the base meal; "Leftover chicken + rice + vegetables" → `### Chicken + Rice + Vegetables` with a note "(also served as leftover lunch)"
- Preserve `+` connectors, slashes, and parentheticals verbatim in the heading
- Strip trailing punctuation
